
# FYP - Xplore

## Develop Environment
>Platform : UE5.1
> 
> Language: C++(MSVC), Blueprint
>

## Control
> W: Move forward
> 
> S: Move backward
> 
> A: Move Left
> 
> D: Move Right
> 
> Q: Throw
> 
> E: Recall
> 
> Tab: Equip / Unequip Weapon
> 
> ESC \ P : Pause
